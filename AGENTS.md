# Habe Dashboard - 原生重装进化版 (AGENTS.md)

## 项目背景
原 “Habe仪表” 微信小程序受限于框架短板（蓝牙并发差、界面刷新滞后、权限风控、看广告等），严重影响了双驱/三驱电动车玩家体验。本项目旨在将小程序的所有核心逻辑迁移至 Android 原生环境，打造极致性能、零广告、全解锁的“性能仪表”。

## 功能模块审计 (Migration Map)

### 1. 仪表核心 (Dashboard)
- **实时监测**: 速度 (内置 GPS / 控制器 RPM 换算)、母线电流、相电流、电压、实时功率。
- **能效统计**: 本次里程、总里程、平均能耗 (Wh/km)、回收能量、瞬时能效。
- **状态报警**: 控温度、电机温度、MOS 故障、电池电压告警。
- **轨迹海报**: 自动记录 GPS 轨迹，生成带能效曲线的骑行分享海报。
- **流体界面**: 自动适配深浅色模式，支持反峰制动时的色彩动态反转 (Regen Dynamic Feedback)。

### 2. 控制器协议矩阵 (Protocols)
| 品牌 | 识别特征 | 核心功能 |
| :--- | :--- | :--- |
| **智科 (Zhike)** | `FFF1/FFF2`, Modbus 0103 | 自动参数读取、转速补偿、系数调校。 |
| **远驱 (Yuanqu)** | `FFE1`, Heartbeat Pulse | 心跳探测、高并发遥测解析。 |
| **APT / 安特 (Ausi)** | `0000FFE1...`, 64-byte Frame | 指令预热、多段特征匹配 (050101...)。 |
| **凌博 (Lingbo)** | `pwd` 握手, 专有序列 | 密码保护连接、马达因子自适应。 |
| **蓝德 (Lande)** | 极客调控版支持 | 动态波特率自适应、信号强度过滤。 |
| **速牛 (MDS)** | 主动轮询模式 | 毫秒级采样率加持，支持加速测试极致精度。 |
| **雁鑫 (Yanxin)** | 密码握手 + 心跳 | 指令流加密处理。 |

### 3. BMS 电池管理系统
- **支持品牌**: 极空 (JK)、蚂蚁 (Ant)、电岭 (Dianling - 含 AES 握手)、明唐、亿纬锂能官方、超威锂电。
- **详情监控**: 单体电压曲线、压差监控、充电/放电 MOS 开关、均衡状态管理。

### 4. 性能工具箱 (Performance Tools)
- **加速测试**: 0-30, 0-50, 0-100km/h 计时，支持重力感应/坡度补偿。
- **制动测试**: 固定时速抱死制动记录。
- **胎压监控 (TPMS)**: 支持九号、极核官方胎压传感器后台实时回传。
- **车库配置**: 车辆多配置文件切换、轮径预设 (inch/mm 转换)、极对数修正。
